# 02. 画面一覧・構成

> **ステータス: 確定（v2 — レビュー反映済み）**

---

## 全体レイアウト: ハイブリッド型

```
┌──────────────────────────────────────────────────────┐
│  ヘッダー  [ロゴ]  [ダッシュボード] [案件] [管理]  [👤]  │
├────────┬─────────────────────────────────────────────┤
│        │                                             │
│ サブナビ │         メインコンテンツ                     │
│（管理系  │                                             │
│ 画面の  │                                             │
│ み表示） │                                             │
│        │                                             │
└────────┴─────────────────────────────────────────────┘
```

**理由:**
- ダッシュボード・一覧画面ではフル幅でコンテンツを広く使える
- 事務局管理画面ではサイドバーで案件管理/応募管理/設定を切り替え
- トップナビでロール共通のグローバル導線を確保
- ロール別にナビ項目が変わる（企業は「投稿管理」、大学/会員は「マイ応募」、事務局は「管理」）

---

## 案件一覧表示: ハイブリッド

- **大学・会員企業向け（U-1）:** カード型（ビジュアル重視、掲示板的な新着感）
- **事務局管理画面（A-2）:** テーブル型（一覧性・操作性重視）
- **企業ダッシュボード（E-1）:** カード型（自社案件を少数表示）

---

## ロール別TOP画面: 分離

ログイン後、ロールに応じた専用ダッシュボードを表示。

---

## MVP画面一覧（全14画面）

### 共通（2画面）

| ID | 画面名 | 概要 |
|----|--------|------|
| C-0 | パブリックTOP（LP） | ログイン前の公開ページ。サービス説明・参加案内・ログイン導線。対外説明にも使える |
| C-1 | ログイン | メール＋パスワード認証 |

### 企業側（3画面）

| ID | 画面名 | 概要 |
|----|--------|------|
| E-1 | 企業ダッシュボード | 自社案件カード一覧、ステータス集計、新規登録ボタン |
| E-2 | ユースケース登録フォーム | 構造化された案件投稿フォーム |
| E-3 | 自社案件詳細 | 投稿内容確認＋応募者一覧＋**応募内容の閲覧**＋ステータス表示 |

### 大学・会員企業側（4画面）

| ID | 画面名 | 概要 |
|----|--------|------|
| U-1 | 案件一覧 | カード型の公開案件一覧＋フィルタ＋**新着表示** |
| U-2 | 案件詳細 | 案件の全情報＋**ワークフローステップ表示**＋応募ボタン |
| U-3 | 応募フォーム | 案件への応募・提案入力 |
| U-4 | マイ応募一覧 | 自分が応募した案件の状況確認 |

### 事務局（5画面）

| ID | 画面名 | 概要 |
|----|--------|------|
| A-1 | 事務局ダッシュボード | KPI集計、直近アクション、ボトルネック表示 |
| A-2 | 案件管理一覧 | テーブル型の全案件管理＋ステータス操作 |
| A-3 | 案件管理詳細 | 個別案件の編集＋応募者管理＋**応募内容閲覧**＋公開制御＋ワークフロー＋アクティビティログ |
| A-4 | 応募管理 | 全応募のテーブル一覧＋対応状況管理 |
| A-5 | **案件代理登録フォーム** | 事務局が企業に代わって案件を登録。企業選択＋E-2同等の入力 |

---

## 画面遷移フロー

```
[C-0 パブリックTOP]
    │
    └──→ [C-1 ログイン]
              │
              ├─ 企業 ────→ [E-1 企業ダッシュボード]
              │                 ├──→ [E-2 登録フォーム] → 投稿完了 → E-1に戻る
              │                 └──→ [E-3 自社案件詳細] → 応募内容確認
              │
              ├─ 大学/会員 → [U-1 案件一覧]
              │                 ├──→ [U-2 案件詳細] → [U-3 応募フォーム] → 完了 → U-1に戻る
              │                 └──→ [U-4 マイ応募一覧] → [U-2 案件詳細]
              │
              └─ 事務局 ──→ [A-1 事務局ダッシュボード]
                               ├──→ [A-2 案件管理一覧] → [A-3 案件管理詳細]
                               ├──→ [A-4 応募管理]
                               └──→ [A-5 案件代理登録] → 完了 → A-2に戻る
```
